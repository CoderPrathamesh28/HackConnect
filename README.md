# HackConnect - The Modern Hackathon Platform 🚀

**HackConnect** is a powerful, lightweight web application designed to streamline the process of organizing, discovering, and participating in hackathons.

## ✨ Features

- **User Authentication**: Secure signup and login with password hashing.
- **Role-Based Access**: 
  - **Admin/Organizer**: Create events, manage submissions, and view the live database.
  - **Participant**: Browse hackathons, join teams, and submit projects.
- **Project Submissions**: Integrated system to submit GitHub links and demo videos.
- **Live Leaderboard**: Real-time listing of latest submissions to keep the competition exciting.
- **Announcements**: Keep participants updated with the latest event news.
- **Profile Customization**: Users can update their bio and social links.

## 🛠️ Tech Stack

- **Backend**: Python 3.x, Flask
- **Frontend**: HTML5, CSS3, Jinja2 (Templating), FontAwesome
- **Database**: SQLite3
